## 🔧 项目一：人力成本分摊分析

### 业务背景
企业需按月按部门核算福利费用，原手工Excel处理10万+行数据耗时3分钟且易出错。

### 核心流程
1. **SQL取数**：从ERP数据库（erp_hr_psndoc）提取员工证件号、所属部门
2. **数据清洗**：从证件号截取生日月份，建立部门性质映射表（销售/管理/生产/研发）
3. **透视分析**：按月份和部门性质统计人数，按每人500元计算福利费
4. **可视化**：使用Pyecharts生成部门费用分布柱状图

🔧 项目二：固定资产更新成本测算
业务背景

企业需定期评估固定资产状态，识别待报废资产并测算更新成本，为预算编制提供依据。
核心流程

    SQL取数：从ERP数据库（erp_fd_asset）提取固定资产台账

    数据筛选：筛选“计算机及电子设备”类别

    折旧率计算：折旧率 = 累计折旧 / 原始成本，标记折旧率>70%的待出售资产

    成本测算：计算待出售资产净值、回收金额（原价10%）、新购预算（原价1.1倍）

关键代码示例
python

# 计算折旧率并识别待出售资产
固定资产台账['折旧率'] = (固定资产台账['累计折旧'] / 固定资产台账['原始成本']).round(2)
待出售资产 = 固定资产台账[固定资产台账['折旧率'] > 0.7]

# 测算更新成本
回收价合计 = (待出售资产['原始成本'] * 0.1).sum()      # 原价10%回收
新购价格合计 = (待出售资产['原始成本'] * 1.1).sum()    # 原价1.1倍新购

成果输出

    生成文件：固定资产台账.xlsx

    控制台输出：待出售资产原值、累计折旧、账面净值、回收价、新购价格
