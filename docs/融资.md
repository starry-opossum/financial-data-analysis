# 企业融资需求测算（Python财务分析）

## 项目背景
基于某化工企业2019-2023年的日度收入数据和年度资产负债表，预测未来一年收入增长率，识别财务敏感项目，测算外部融资需求。

## 我的工作
1. **SQL取数**：从数据库提取收入明细（1825行）
2. **时间序列预测**：使用霍尔特温特模型（加法趋势+季节）预测下一年度收入，计算增长率
3. **财务敏感性分析**：计算资产负债表各项目与收入的相关系数，标记敏感项目（|r|>0.8）
4. **融资需求测算**：预测敏感项目金额，计算融资总需求 → 内部融资（净利润）→ 外部融资缺口
5. **可视化**：绘制资产总额、负债总额、营业收入趋势面积图（pyecharts）

## 技术栈
- SQL（数据提取）
- Python：Pandas, statsmodels（ExponentialSmoothing）, pyecharts
- 财务概念：敏感项目、融资总需求、内外部融资

## 关键结论
- 预测收入增长率：23%
- 外部融资需求：151960001.90507元
- 资产、负债、收入均呈上升趋势，需关注流动性风险
